# Praetoria Warfront

Praetoria Warfront is a digital, Risk-inspired strategy game set on the world map of Praetoria from Splinterlands.

This build is a static HTML version, so it is easy to share and test with friends.

## What's in this repo

- `index.html` - the playable game
- `praetoria-map-no-numbers-board.png` - the cleaned map art used by the board
- `assets/audio/` - placeholder audio folder for future real sound assets

## How to play locally

1. Download or clone the repository.
2. Open `index.html` in a browser.
3. If your browser blocks local assets, serve the folder with any simple local web server and open that address instead.

## Current features

- Splinterlands-flavored commanders and regions
- Mount Praetorous and The Secret of Praetoria win pressure
- Objective selection
- AI opponents
- Bjangles mode for a harder ruleset
- Mobile-conscious map controls and zoom tools

## Sound note

The game currently includes fallback generated sound effects in code. The `assets/audio` folder is ready for real `.mp3` files if we add them later.

## Publishing

Because this is a static game, it is a good fit for GitHub Pages.

Typical setup:

1. Push this repository to GitHub.
2. In the repository settings, enable GitHub Pages from the main branch.
3. Use the site URL GitHub gives you to share the game.
