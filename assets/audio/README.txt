Drop real audio files in this folder to replace the built-in synthesized sounds.

Supported filenames:
- select.mp3
- deploy.mp3
- turn.mp3
- fortify.mp3
- expand.mp3
- attack.mp3
- capture.mp3
- secret.mp3
- eliminate.mp3
- victory.mp3

Notes:
- MP3 is expected by the current game file.
- If a file is missing, the game falls back to the built-in synthetic sound for that event.
- Short files work best for sound effects.
